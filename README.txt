Pré-requis : Avoir installer Nodejs

1. Ouvrir l'invite de commande et aller dans le dossier ./back

2. Taper npm i dans l'invite de commande et attendre la fin de l'installation

3. Lancer l'application avec la commande npm start dans le même chemin

4. Aller dans le dossier ./front avec l'invite de commande

5. Taper npm i dans l'invite de commande et attendre la fin de l'installation

6. Lancer l'application avec la commande npm start dans le même chemin
